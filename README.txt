Pour installer le projet, clonez le repo, installez mysql vous pouvez ensuite créer la base de donnée gm__db depuis le fichier ./back/db/gm__db--struct
créez un .env depuis les .env_examples dans le dossier /back & dans le dossier frontend/globomania/ vous aurez besoin d'une clé API provenant de tenor GIF
https://tenor.com/gifapi

lancez le back => cd back puis nodemon start

lancez le front => cd frontend puis npm start
